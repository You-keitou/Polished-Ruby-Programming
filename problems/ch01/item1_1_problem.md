# 第1章 1.1: いつ組み込みクラスを使うべきか

## 問題：呪われたハッシュ (The Haunted Hash)

あなたは言葉をカテゴリ（例：'verbs', 'nouns'）ごとに整理する `WordCategorizer` クラスを作成しています。
しかし、現在の実装には重大なバグがあります。あるカテゴリに単語を追加すると、まだ触れていないすべてのカテゴリにもその単語が現れてしまうようです！

### 課題

1.  テストを実行して失敗を確認する：`bundle exec rspec spec/ch01/item1_1_spec.rb`
2.  `lib/ch01/item1_1.rb` を分析し、なぜ `Hash` が予期しない動作をするのか理解する。
3.  `WordCategorizer` の `initialize` メソッドを修正し、各カテゴリが独立した単語リストを持つようにする。

### ヒント
ハッシュのデフォルト値がどのように定義されているか注意深く見てください。Rubyでは、`Hash.new([])` はすべての存在しないキーに対して *同じ* 配列オブジェクトをデフォルトとして返します。
